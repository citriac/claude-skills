## Serverless 平台对比

### Cloudflare Workers
- **调用量**：10万次/天（全球最佳）
- **冷启动**：极快（<5ms）
- **定价**：超额 $0.15/百万次请求
- **限制**：每个 Worker 10ms CPU 时间/请求
- **优势**：全球边缘网络，免费额度高

### Vercel Functions
- **调用量**：10万次/月
- **冷启动**：较快（通常 <200ms）
- **定价**：超额按使用量计费
- **限制**：10秒超时，50MB 包大小
- **优势**：Next.js 深度集成，开发体验好

### Netlify Functions
- **调用量**：12.5万次/月
- **冷启动**：一般（可能 >1秒）
- **定价**：超额 $25/百万次请求
- **限制**：10秒超时
- **优势**：框架灵活，集成 Netlify 生态

### Deno Deploy
- **调用量**：10万次/月
- **冷启动**：极快（<50ms）
- **定价**：超额按请求和计算时间计费
- **限制**：无明确超时限制
- **优势**：Deno 原生，TypeScript 支持好

## 注意事项

### 免费层限制
1. **资源限制**：注意请求数、存储、带宽等硬限制
2. **功能限制**：某些高级功能可能需要付费
3. **时间限制**：试用期通常 1-12 个月
4. **活动要求**：有些服务要求定期活动避免暂停

### 成本控制
1. **设置预算告警**：所有付费服务设置预算告警
2. **监控使用量**：定期检查资源使用情况
3. **清理闲置资源**：定期清理不再使用的资源
4. **理解定价模式**：清楚了解超额费用计算方式

### 技术考虑
1. **供应商锁定**：避免过度依赖单一供应商
2. **迁移成本**：考虑未来迁移到其他平台的成本
3. **合规要求**：考虑数据存储和处理的法律要求
4. **性能需求**：根据用户分布选择合适的地理位置
